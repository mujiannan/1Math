# 简介
这是一个针对工作中特殊需求而开发的VSTO for Excel项目，完全个人独立开发。
# 实现功能
1. 批量检测链接是否能正常打开
2. 批量检测流媒体的信息，包括时长、分辨率等
3. 批量取消合并的单元格并智能填充
4. 批量翻译
5. 批量生成二维码
# 开发后记
功能实现不难，优化性能花费了很大的力气，VSTO的性能挖掘一下还是很令人满意的~
1. 运用多线程，并写下了专用于excel的数据并行，达到极高的运行效率，其中“取消合并单元格功能”功能的速度超过行业知名插件“Excel催化剂”40%，“批量翻译”功能的速度超过300条目/秒，“批量生成二维码”功能的速度在固态硬盘下超过100个/秒；
2. 运用任务异步，实现了优秀的界面响应；
